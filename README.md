# MMC
Este projeto apresenta um algoritmo simples que calcula o MMC de um conjunto de números; Primeiro é descoberto o MDC (máximo divisor comum) pelo Algoritmo de Euclides e logo após é feito o cálculo para descobrir o MMC (menor múltiplo comum);

## 🛠️ Tecnologias e Conceitos 
Python 3.x e Lógica Matemática - Algoritmo de Euclides e Menor Múltiplo Comum📋

### Pré-requisitos: 
Python instalado; Git; Gerenciador de pacotes (pip);
