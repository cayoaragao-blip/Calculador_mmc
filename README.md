# MMC
Este projeto apresenta um algoritmo simples que calcula o MMC de dois números; Primeiro é descoberto o MDC (máximo divisor comum) e logo então é feito o cálculo para descobrir o MMC (menor múltiplo comum)
mmc=(n1*n2)/mdc;

## 🛠️ Tecnologias e Conceitos 
Python 3.x e Lógica Matemática - Algoritmo de Euclides📋

### Pré-requisitos: 
Python instalado; Git; Gerenciador de pacotes (pip);
